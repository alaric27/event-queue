# event-queue
基于事件驱动的内存级消息队列
为了学习disruptor，所以写了这个项目。
本项目大量参考和直接使用disruptor源码

由于disruptor 相关概念难以理解，且各组件直接依赖关系复杂
所以抽象了下面几个组件：
## producer 
生产者，用于消息发送

## consumer
消费者，用于消息接收

## broker
中介，用于消息存储及协调生产者和消费者
